# AWS Project Cleanup Guide 🧹

Follow these steps AFTER your hackathon evaluation to ensure your AWS account stays at **Zero Cost**.

## Step 1: Delete AWS Amplify App
1. Go to the **AWS Amplify Console**.
2. Select **Vidya-Setu Frontend**.
3. Under **Actions**, choose **Delete app**.
   * *This stops hosting and CDN costs.*

## Step 2: Delete AWS Lambda & API Gateway
1. Go to **Lambda Console**.
2. Find your function (e.g., `VidyaSetu-Backend`).
3. Choose **Actions** -> **Delete**.
4. Go to **API Gateway Console**.
5. Find the API created for the backend and **Delete** it.

## Step 3: Delete Amazon DynamoDB Table
1. Go to **DynamoDB Console**.
2. Click **Tables** -> Select `VidyaSetu_Progress`.
3. Click **Delete**.
   * *This removes all student progress data and caches.*

## Step 4: Revoke IAM Access Keys
1. Go to **IAM Console**.
2. Click **Users** -> Your User.
3. Go to **Security Credentials**.
4. **Deactivate** or **Delete** the Access Keys you created for the project.
